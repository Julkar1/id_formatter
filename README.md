# id_formatter
ID Formatter is a lightweight, browser-based tool for converting and formatting numeric IDs. It supports Bangla → English digit conversion, zero-padding, duplicate cleaning, and exporting the results into multiple formats (TXT split files, Excel, CSV, SQL). No external dependencies—just pure HTML, CSS, and JavaScript running entirely offline.
